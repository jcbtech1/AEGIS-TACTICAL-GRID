
# Aegis Tactical Grid (Frontend Edition)

Este es el proyecto **Aegis Tactical Grid**, una interfaz de comando táctico avanzada construida con Next.js 15, React, Tailwind CSS y Genkit AI.

## Características
- **Consola Táctica**: Monitoreo de red en tiempo real simulado (100% Frontend).
- **IA Integrada**: Chat táctico y análisis de seguridad impulsado por Gemini.
- **Diseño Inmersivo**: Estética militar "Cyber-Grid" con efectos CRT y escaneo láser.
- **Optimizado**: 100% funcional como aplicación Frontend autónoma, ideal para Replit.

## Inicio Rápido
1. Instala dependencias: `npm install`
2. Configura `.env`: `GOOGLE_GENAI_API_KEY=tu_llave`
3. Inicia: `npm run dev`

Para más detalles sobre la operación y el despliegue en Replit, consulta `README_LOCAL.md`.
